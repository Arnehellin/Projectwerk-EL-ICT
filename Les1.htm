Les 1: Uitleg HTML, CSS en opgaves

In de eerste les leerden we websites maken met behulp van HTML en CSS.

HyperText Markup Language, of kortweg HTML, is een opmaaktaal voor websites. Een webbrowser kan documenten die geschreven zijn in HTML lezen en daarna weergeven op het scherm.

CSS staat voor Cascading Style Sheet en wordt gebruikt om webpaginaâ€™s vorm te geven. Ieder element kan afzonderlijk bepaald worden aan de hand van CSS. Zo kunnen we het kleur, het lettertype, het achtergrondkleur en nog veel andere kenmerken van de tekst veranderen. De CSS- file wordt dan toegevoegd aan de HTML-code. Wanneer er een element veranderd wordt in het CSS bestand, zal deze automatisch worden doorgegeven aan de HTML-code.

We kregen de opdracht om een website te maken en deze daarna te publiceren op het internet. Eerst moesten we beter leren werken met HTML. Het was mogelijk om online uitleg te vinden over hoe we kunnen coderen met HTML. Aan de hand van oefeningen ondervonden we hoe we gebruik konden maken van hyperlinks en hoe we een CSS bestand konden toevoegen aan onze website.
